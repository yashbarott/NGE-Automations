# NGE Automations Website

### NextGenEnterprise Automations LLC

---

## 📌 Project Overview

This project represents the official website for **NGE Automations**, an enterprise consulting firm focused on automation-driven transformation.

The website is designed to:

* Establish enterprise-level credibility
* Showcase automation consulting capabilities
* Drive consultation inquiries from global clients

---

## 🎯 Core Objectives

* Position the brand as a **trusted enterprise consulting partner**
* Highlight **automation strategy & system integration expertise**
* Provide a **clear conversion path** via consultation form

---

## 🎨 Design System

### Color

* Primary: Enterprise Blue
* Background: White / Light Gray
* Accent: Minimal dark tones

---

### Typography

* Font Family: **Inter**
* Usage:

  * Headings → Medium
  * Body → Regular
* No unnecessary bold styling

---

### UI Style

* Clean & minimal
* Enterprise-focused
* Structured layout
* High whitespace usage

## 📐 Layout System

* Grid: 12-column
* Container Width: ~1200px
* Spacing: Generous vertical rhythm

---

## 🚫 Design Restrictions

* No multi-color logo usage
* No bold emphasis on partial words (e.g., "NextGen")
* No gradients or shadows on logo
* Avoid stock-heavy imagery
* Prefer system/automation visuals
